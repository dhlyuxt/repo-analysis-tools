# Repo Analysis Tools

Codex Quickstart

This repository is the daily-use entry point for Codex-based C repository analysis.

1. Put your local Codex settings in `.codex/config.toml`.
2. Keep the MCP wiring in `.mcp.json` pointed at `repo-analysis-mcp`.
3. Use `/home/hyx/anaconda3/envs/agent/bin/python tools/run_self_use_demo.py` to validate the launch path on a real fixture.

The project stays self-use focused: scan, understand, trace impact, render Markdown, and export reusable bundles.
